
Дамен верхняга ўзроўню (англ.: top-level domain, скар. TLD) — апошняя частка назвы дамена, паслядоўнасць літар пасля апошняй кропкі ў назве любога дамена. Так, у назве дамена www.website.com, даменам верхняга ўзроўню з'яўляецца com (альбо COM, таму што падзелу на вялікія і малыя літары ў назвах даменаў няма).

Арганізацыя Internet Assigned Numbers Authority (IANA) на сённяшні дзень падзяляе дамены верхняга ўзроўню на тры тыпы:

Поўны спіс даменаў верхняга ўзроўню змешчаны ў наступным артыкуле: спіс даменаў верхняга ўзроўню.

Дамен .nato быў утвораны ў канцы 1980-х арганізацыяй NIC для НАТА, але праз некаторы час высветлілася, што ні адзін з такіх даменаў не дазваляў адэкватна вызначыць статус міжнародных арганізацый. Таму неўзабаве NIC быў утвораны яшчэ адзін дамен з назвай .int для ўжывання міжнароднымі арганізацыямі, прапанаваўшы НАТА выкарыстоўваць замест існуючага варыянта nato.int. Такім чынам, дамен верхняга ўзроўню nato перастаў выкарыстоўвацца, але канчаткавае яго выдаленне адбылося толькі ў ліпені 1996 года.

Іншымі гістарычнымі даменамі верхняга ўзроўню былі .cs для Чэхаславакіі, .su для Савецкага Саюза і .zr для Заіра.
